Tentang Aplikasi

Pandemi Covid-19 sangat berdampak di semua lini kehidupan.sosial, ekonomi, politik, budaya dan pendidikan. Sekolah di liburkan, di ganti dengan sistem pembelajaran jarak jauh (Daring).

Aplikasi ini di harapkan bisa menjadi salah satu solusi untuk menjembatani antara guru dan murid agar tetap terhubung di tengah pandemi saat ini.

melalui aplikasi ini di harapkan bisa mempermudah guru untuk memberikan tugas kepada siswa, dan begitupun sebaliknya, siswa dengan mudah mengirimkan hasil tugas yang di berikan oleh guru. aplikasi ini menggunakan tehnologi PWA (Progressive Web App), dimana fitur unggulan dari tehnologi ini adalah siswa ataupun guru tetap bisa menggunakan aplikasi ini di saat tidak terhubung di jaringan sekalipun. dan tetap bisa mengirim file ataupun lainnya, dan akan di kirimkan ke server pada saat user telah terhubung kembali di jaringan tanpa khawatir kehilangan konten yang sudah di buat.
